probrable algorithm :
1. Parse OCI Configuration
   - Read config.json specification
   - Validate container parameters
   - Prepare runtime environment

2. Setup Filesystem
   - Create container root filesystem
   - Mount necessary directories
   - Setup copy-on-write layers

3. Create Namespaces
   - Fork new process
   - Create required namespaces (PID, mount, network, etc.)
   - Configure namespace-specific settings

4. Configure Cgroups
   - Create cgroup hierarchy
   - Set resource limits and constraints
   - Attach container process to cgroups

5. Setup Security
   - Apply security profiles (SELinux, AppArmor)
   - Configure capabilities and seccomp filters
   - Set user/group mappings

6. Execute Container Process
   - Change root directory (chroot)
   - Setup environment variables
   - Execute container's main process

7. Monitor and Manage
   - Track process lifecycle
   - Handle signals and cleanup
   - Collect logs and metrics
